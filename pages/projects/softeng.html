Hospital Application

March 2021

Overview

Developed by a team of 10, this comprehensive hospital application is designed to create a more connected and efficient environment for patients, visitors, and staff. Built on a robust SQL database, the app centralizes a range of key hospital functions, from internal navigation to streamlined service management, all accessible through a single, intuitive platform.

The app's core feature is intelligent pathfinding. Leveraging an A* (A-star) algorithm, the wayfinding feature allows anyone to effortlessly navigate the hospital's five floors. After displaying the most efficient route, users can export it as a CSV file for easy offline access.

In addition to navigation, the app provides a user-centric service request portal for patients and guests, with requests logged into the database for assignment. For employees, a real-time notification system ensures they are instantly alerted to new or incomplete tasks on their main page, improving workflow and response times.

Key Features

  • Feature 1: A Algorithm-based Pathfinding
  • Feature 2: SQL Database
  • Feature 3: Service Request Portal
  • Feature 4: Real-Time Task Notifications
  • Feature 5: Chatbot
