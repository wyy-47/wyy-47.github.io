Multimodal AI Robotic System for ASD Therapy

Started: January 2025 | Status: In Progress

Project Overview

The ultimate goal of this project is to deploy Nao (or other humanoid robot) to conduct therapeutic session with Autism kids through movements. Now we focus on emotion learning.

Each learning session will contain 4 emotions, and 5 games. We wanted to make each section of the session fun and game like, with verbal rewards from Nao. The 5 games are progressive, gradually increase interaction complexity, using multimodal cues (verbal, facial, bodily).
